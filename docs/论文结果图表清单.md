# 论文结果图表清单

## 必备结果图

| 图 | 来源脚本/目录 | 用途 |
| --- | --- | --- |
| 三链路总体架构图 | 手工绘制，参考 `src/link/default_link_spec.m` | 说明统一 Tx + 三套 Rx 的模块化设计 |
| 帧结构图 | 手工绘制，参考会话控制、前导、物理头、payload | 说明同步、控制面和数据面 |
| BER vs Eb/N0 | `sweep_profile_ebn0.m` | 展示三条链路随信噪比变化的可靠性 |
| rawPER/PER vs Eb/N0 | `sweep_profile_ebn0.m` | 区分外码修复前后性能 |
| 窄带中心频点扫描图 | `scan_narrowband_dsss_fh_centers.m` | 证明不是单一频点偶然通过 |
| 多径参数扫描柱状图 | `validate_three_links_6db.m` 或 `scan_rayleigh_multipath_cases.m` | 展示 SC-FDE + 2Rx MRC 的稳定性 |
| 脉冲概率扫描图 | `validate_three_links_6db.m` 或 `scan_impulse_cases.m` | 展示 blanking 在 JSR 标定口径下有效 |
| Bob/Eve 错钥恢复对比 | `sweep_profile_ebn0.m` 开启 `EnableEve=true` | 支撑抗破解分析 |
| Warden Pe/Pd/Pfa 曲线 | `sweep_profile_ebn0.m` 开启 `EnableWarden=true` | 支撑低可检测性分析 |

## 必备结果表

| 表 | 关键字段 |
| --- | --- |
| 三链路默认参数表 | profile、调制、FH频点数、DSSS、SC-FDE、2Rx MRC、RS配置、burst |
| 三链路 6 dB 验收表 | profile、干扰参数、method、rawPER、PER、burstSec、elapsedSec、pass |
| Eb/N0 曲线数据表 | profile、method、Eb/N0、JSR、BER、rawPER、PER、PSNR、SSIM |
| 窄带扫频表 | center、bandwidth、method、rawPER、PER、headerSuccess、elapsedSec |
| 多径扫描表 | delays、gains、rawPER、PER、frontEnd/header/session/payload success |
| 脉冲扫描表 | impulseProb、JSR、method、rawPER、PER、elapsedSec |
| Eve 对比表 | role、method、Eb/N0、BER、PER、PSNR、payloadSuccess |
| Warden 检测表 | layer、bobEbN0dB、wardenEbN0dB、Pd、Pfa、Pmd、Pe、xi |

## 推荐最终运行命令

```matlab
addpath(genpath('src'));
sweep_profile_ebn0( ...
    'Profiles', ["impulse" "narrowband" "rayleigh_multipath"], ...
    'EbN0List', [0 2 4 6 8 10], ...
    'JsrDb', 0, ...
    'NFramesPerPoint', 3, ...
    'EnableEve', true, ...
    'EnableWarden', true, ...
    'WardenTrials', 200, ...
    'MakeFigures', true, ...
    'ExportTables', true, ...
    'Tag', 'thesis_final');
```

## 写作边界

- 可以写：三条单干扰链路均已在统一口径下完成端到端验证。
- 可以写：传统方法在当前配置下比机器学习方法更稳定。
- 可以写：Eve/Warden 作为扩展评估层接入，不参与 Bob 主链路决策。
- 不建议写：机器学习方法全面优于传统方法。
- 不建议写：系统能抵抗任意混合干扰。
- 不建议写：系统绝对不可截获或不可检测。
