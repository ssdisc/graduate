# 三链路验收规范表

## 统一口径

- 主链路只验收 `impulse`、`narrowband`、`rayleigh_multipath` 三个 profile。
- 默认输入载荷为灰度图，长边 `256`，短边按比例缩放。
- 统一验证点为 Bob 端 `Eb/N0=6 dB`、`JSR=0 dB`。
- 完整链路必须包含：同步、公共物理头/会话控制、payload 解码、outer RS、图像恢复。
- `rawPER` 定义为 outer RS 修复前有效数据包失效率；`PER` 定义为 outer RS 修复后有效数据包失效率。
- 硬性通过原则：对规定参数组合逐点验证，所有组合都要满足 `PER=0`。
- 时间约束：`burstDurationSec < 60 s`，单帧仿真时间目标 `< 60 s`。

## 分链路验收

| 链路 | 主抗扰机制 | 参数覆盖 | 硬性指标 |
| --- | --- | --- | --- |
| `impulse` | 混沌 FH + 同步/公共头 + `blanking/clipping` + 交织 + 内码 + outer RS | `impulseProb={0.01,0.03,0.05}`，按平均干扰功率标定 `JSR=0 dB` | 所有组合 `PER=0`，`burst<60 s` |
| `narrowband` | 混沌 FH + DSSS + 受保护控制面 + FH 域窄带估计/擦除 + outer RS；不启用 payload diversity | 基础扫频 `center=-3:0.5:3`，`bw=1.0`；扩展抽检 `bw={0.5,1.0,1.5}`、`center={-2.5,0,2.5}`，并满足 `|fc|+bw/2<0.5` | 所有组合 `PER=0`，`burst<60 s` |
| `rayleigh_multipath` | 混沌 FH + 同步/前导 + SC-FDE + `2Rx MRC` + 内码 + outer RS | `([0 1 2],[0 -6 -10])`、`([0 2 4],[0 -6 -10])`、`([0 1 4],[0 -8 -14])`、`([0 3 4],[0 -6 -12])` | 所有组合 `PER=0`，`burst<60 s` |

## 研究型附加检查

- `impulse`：在 `(0.03,50)` 与 `(0.05,80)` 上，主方法相对 `none` 的 `rawPER` 应显著下降。
- `narrowband`：`fh_erasure` 相对 `none` 应在每个已测中心频点上改善 `rawPER`。
- `rayleigh_multipath`：`2Rx` 分支级 SC-FDE 合并相对 `1Rx` 不得恶化 `rawPER`，并以降低外码依赖为目标。
