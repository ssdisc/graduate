# 毕业设计（论文）开题报告

**课题名称**：基于跳频与抗干扰调制的隐蔽图像通信系统设计与实现

---

## 一、选题的背景和意义

### 1.1 选题背景

在当今数字化以及信息化的浪潮当中，无线通信技术已然成为连接物理世界与数字世界的枢纽。然而，随着无线设备、个人移动终端的爆炸式增长以及电磁频谱的日益拥挤，通信环境正变得前所未有的复杂。特别是在军事国防、公共安全以及关键基础设施监控等敏感领域，信息的传输安全不再仅仅局限于数据内容的保密，更延伸到了通信行为本身的隐蔽以及链路的抗干扰的能力。

近年来，深度学习技术在无线通信物理层的应用取得了突破性进展。O'Shea等人[13]首次提出将通信系统视为端到端自编码器进行联合优化，开创了深度学习在物理层通信中的应用先河。Ye等人[14]证明了深度神经网络在OFDM系统信道估计和信号检测中的强大能力，尤其在非理想信道条件下表现出比传统MMSE估计器更强的鲁棒性。Shlezinger等人[15]系统性地总结了基于模型的深度学习方法，为将领域知识与数据驱动方法相结合提供了理论框架。在图像传输领域，Bourtsoulatze等人[19]提出了深度联合信源信道编码（Deep JSCC）方法，将图像压缩与信道编码统一为端到端神经网络，在无线信道下实现了优雅降级的图像传输性能。Wang等人[20]进一步引入注意力机制，提出ADJSCC方案，显著提升了图像传输的质量和效率。这些研究为本课题采用机器学习方法进行脉冲噪声抑制和接收端图像增强提供了理论基础和技术参考。

### 1.2 选题意义

本课题"基于跳频与抗干扰调制的隐蔽图像通信系统设计与实现"旨在把混沌理论、跳频通信、抗干扰调制技术、隐蔽通信技术以及深度学习技术等进行深度融合，来构建一个鲁棒、安全、隐蔽的无线图像传输原型系统。它的意义主要体现在以下几个方面：

#### 1.2.1 理论层面的创新意义

本研究将探索运用混沌映射，像Logistic映射等的非线性动力学特性来生成跳频序列。相较于传统伪随机序列，混沌序列具有对初值极端敏感、非周期性、宽带类噪声等特性。研究混沌序列在有限精度下的退化问题以及它在跳频系统当中的统计特性，比如均匀性、汉明相关性，将为提高扩频通信的抗截获能力提供新的理论支撑。

#### 1.2.2 工程应用层面的实践价值

本研究设计的抗干扰跳频方案可应用于对安全和隐蔽性要求较高的场合，例如在智能电网、核电站等关键设施的无线监控网络当中，工业电磁干扰严重并且存在非法入侵的风险。本课题提出的混沌跳频方案能有效抵抗工业噪声干扰，防止视频流被恶意篡改或者窃取，为工业物联网安全提供技术参考。

#### 1.2.3 机器学习与通信融合的前沿意义

本研究将探索深度学习技术在抗脉冲干扰通信系统中的应用。相较于传统的固定阈值blanking和clipping方法，基于神经网络的脉冲检测方法具有以下优势：

1. **自适应性强**：神经网络能够从数据中学习脉冲噪声的统计特性，自动适应不同的信道环境
2. **软判决输出**：深度学习模型可输出软可靠性权重，与Viterbi软判决译码结合实现更优的纠错性能
3. **端到端优化**：借鉴基于模型的深度学习框架[15]，将领域知识与数据驱动方法相结合，兼顾可解释性与性能
4. **图像后处理增强**：借鉴Deep JSCC[19][20]的思想，在接收端利用深度学习进行图像去噪和质量增强

## 二、研究的基本内容和拟解决的主要问题

本课题旨在设计实现一个基于跳频以及抗干扰调制的隐蔽图像通信系统。系统应能在加性高斯白噪声信道，以及存在人为干扰如单音干扰、窄带干扰的恶劣无线信道环境当中，实现图像信息的低误码率传输，同时具备低截获概率特性。研究的内容和拟解决的主要问题可以大致划分为以下几个任务：

### 2.1 任务一：图像信源处理与混沌加密策略

本部分主要解决图像数据的高效压缩以及安全防护问题。对于前者，鉴于跳频信道带宽有限，需对原始图像如512x512 RGB进行灰度化处理，并且研究采用离散余弦变换来开展有损压缩，去除人眼不敏感的高频分量，降低数据传输率，从而提高扩频增益。对于后者，考虑设计混沌加密算法，先对图像像素坐标来开展多轮迭代置乱，打破图像的空间邻域相关性，再运用Logistic映射或者Henon映射生成的混沌序列来当作密钥流，与置乱后的像素进行按位异或运算。借助引入密文反馈机制，使得任一像素的微小变化都能扩散至整个图像，极大程度上提高抗差分攻击能力。

### 2.2 任务二：混沌跳频通信系统架构设计

本部分是核心通信链路的设计，涵盖序列生成以及调制过程。预计将设计混沌跳频序列生成器（CS-FHSS）、抗干扰调制器以及跳频发射机，研究不同的混沌映射算法的性能，对比分析BPSK、QPSK、MSK等调制方案在跳频系统当中的频谱效率与抗干扰性能，设计频率合成器模型，来模拟跳频的载波快速切换过程。

### 2.3 任务三：信道建模与接收机关键技术

本部分模拟真实的电磁对抗环境并且设计接收算法。通过构建AWGN信道来模拟热噪声，构建瑞利衰落信道来模拟多径效应，构建干扰模型包括单音干扰、部分频带噪声干扰，以及扫频干扰等常见的干扰信号用来测试系统的抗干扰极限。对于接收机的设计，需要实现同步以及解调，预计将利用混沌序列的自相关尖峰特性来实现粗同步，设计延迟锁定环也就是DLL来维持时钟对齐，研究在接收端采用FFT变换域滤波或者自适应陷波器，剔除明显的窄带干扰信号，提高解调前的载干比。

### 2.4 任务四：系统仿真与综合性能评估

搭建基于MATLAB/Simulink的完整仿真平台，绘制不同调制方式、不同干扰强度下的BER曲线来分析误码率，计算解密图像的MSE、PSNR、SSIM，并结合主观视觉感知来进行图像质量评价，对比跳频信号以及背景噪声的统计特征，计算KL散度作为隐蔽性度量指标以评价隐蔽性。最终考虑三种指标，选择综合性能较好的方案作为最终采用的方案。

### 2.5 任务五：基于深度学习的脉冲噪声检测与抑制

本部分是系统的创新点之一，旨在利用机器学习方法提升系统在脉冲干扰环境下的性能。

#### 2.5.1 特征工程设计

参考Zhang等人[16]在图像去噪领域的残差学习思想，设计适用于通信信号的特征提取方案：

| 特征 | 描述 | 物理意义 |
|------|------|----------|
| $\|r\|$ | 接收符号幅度 | 脉冲噪声导致幅度异常增大 |
| $\|r\|/\text{median}(\|r\|)$ | 归一化幅度 | 消除信号功率变化的影响 |
| $\|\|r_i\| - \|r_{i-1}\|\|$ | 幅度差分 | 检测突发性幅度变化 |
| $\angle r$ | 相位信息 | 脉冲噪声可能导致相位跳变 |

#### 2.5.2 神经网络模型设计

**1D卷积神经网络（CNN）模型**：

借鉴DnCNN[16]的残差学习架构，设计用于脉冲检测的1D CNN：

```
输入(4特征) → Conv1D(16滤波器,k=5) → BN → ReLU
           → Conv1D(32滤波器,k=3) → BN → ReLU
           → FC(4输出)
```

输出包括：
- 脉冲概率 $p \in [0,1]$
- 可靠性权重 $w \in [0,1]$
- 清洁符号估计 $(\hat{r}_{\text{real}}, \hat{r}_{\text{imag}})$

**门控循环单元（GRU）模型**：

利用GRU捕捉接收符号序列的时序相关性：

```
输入(4特征) → GRU(hiddenSize=32) → FC(4输出)
```

GRU模型能够利用上下文信息，对连续脉冲干扰具有更好的检测能力。

#### 2.5.3 训练策略

- **数据生成**：利用仿真链路生成带标签的训练数据，标签为脉冲发生位置
- **损失函数**：采用二元交叉熵损失（脉冲检测）+ MSE损失（符号估计）的加权组合
- **在线/离线结合**：参考Ye等人[14]的方法，离线训练模型，在线进行推理

#### 2.5.4 与传统方法的对比

| 方法 | 原理 | 软输出 | 自适应性 |
|------|------|--------|----------|
| Blanking | 幅度超阈值置零 | 否 | 差 |
| Clipping | 幅度超阈值削波 | 否 | 差 |
| ML-Blanking | 逻辑回归检测+置零 | 否 | 中 |
| **ML-CNN** | CNN检测+软抑制 | **是** | **强** |
| **ML-GRU** | GRU检测+软抑制 | **是** | **强** |

### 2.6 任务六：基于深度学习的接收端图像增强

本部分是系统的另一创新点，旨在利用深度学习技术对接收端重建的图像进行后处理增强，进一步提升图像质量。

#### 2.6.1 研究动机

即使经过信道编码和脉冲抑制，接收端重建的图像仍可能存在以下问题：
- 残余比特错误导致的像素噪点
- 脉冲干扰造成的局部图像损坏
- 信道衰落引起的图像质量下降

传统方法难以有效处理这些复杂的图像退化问题，而深度学习在图像去噪和恢复领域已展现出卓越的性能。

#### 2.6.2 技术方案

借鉴Deep JSCC[19]和ADJSCC[20]的思想，设计接收端图像增强模块：

**方案一：基于DnCNN的图像去噪**

采用Zhang等人[16]提出的DnCNN架构进行残差学习：

```
输入图像 → Conv(64,k=3) → [Conv(64,k=3) → BN → ReLU] × 15 → Conv(1,k=3) → 残差输出
                                                                    ↓
                                                          输入图像 - 残差 = 去噪图像
```

**方案二：基于U-Net的图像恢复**

对于严重损坏的图像，采用编码器-解码器结构：

```
编码器：逐层下采样提取多尺度特征
解码器：逐层上采样恢复图像细节
跳跃连接：保留高频信息
```

#### 2.6.3 训练数据构建

- **输入**：经过通信链路传输后的含噪/损坏图像
- **标签**：原始发送图像
- **数据增强**：不同SNR、不同脉冲概率、不同图像内容

#### 2.6.4 损失函数设计

采用多目标损失函数：

$$\mathcal{L} = \lambda_1 \mathcal{L}_{MSE} + \lambda_2 \mathcal{L}_{SSIM} + \lambda_3 \mathcal{L}_{perceptual}$$

其中：
- $\mathcal{L}_{MSE}$：像素级均方误差，保证数值精度
- $\mathcal{L}_{SSIM}$：结构相似性损失[11]，保持图像结构
- $\mathcal{L}_{perceptual}$：感知损失，提升视觉质量

#### 2.6.5 系统集成

图像增强模块在接收端的位置：

```
接收符号 → 脉冲抑制(ML) → 软解调 → Viterbi译码 → 解扰码 → 帧解析
                                                          ↓
                                                     图像重建
                                                          ↓
                                               ★ CNN图像增强 ★
                                                          ↓
                                                     最终输出图像
```

#### 2.6.6 预期效果

| 指标 | 无增强 | 有增强（预期） |
|------|--------|----------------|
| PSNR | 基准值 | +2~5 dB |
| SSIM | 基准值 | +0.05~0.15 |
| 主观质量 | 可见噪点 | 噪点明显减少 |

## 三、研究方法及措施

本课题将遵循理论研究、算法仿真、系统集成、性能评估的技术路线：

### 3.1 理论研究

深入研究混沌动力学方程、扩频通信原理、数字图像处理、密码学基础以及深度学习理论等。

### 3.2 算法仿真

运用MATLAB分别编写混沌发生器、加密模块、跳频调制器、信道模型、机器学习脉冲抑制模块、图像增强模块等子程序模块，分别验证其可行性。

### 3.3 系统集成

在Simulink当中或者直接编写MATLAB脚本连接各模块，形成闭环通信链路，验证整个系统的可行性。

### 3.4 性能评估

搭建仿真平台，模拟干扰源以及窃听者，对系统来开展压力测试以及隐蔽性和图像质量验证，并对子程序可以采用的不同算法进行对比分析，综合评判后选择最佳方案。

### 3.5 机器学习方法

本课题将采用以下机器学习技术路线：

1. **数据准备**：利用MATLAB仿真平台生成大量带标签的训练样本，涵盖不同SNR、不同脉冲概率的场景

2. **模型设计**：
   - 参考O'Shea等人[13]的端到端学习思想
   - 借鉴Zhang等人[16]的残差学习和批归一化技术
   - 采用Shlezinger等人[15]提出的基于模型的深度学习框架
   - 参考Bourtsoulatze等人[19]和Wang等人[20]的Deep JSCC方法

3. **模型训练**：
   - 使用MATLAB Deep Learning Toolbox进行模型训练
   - 采用Adam优化器，学习率自适应调整
   - 使用早停策略防止过拟合

4. **性能评估**：
   - 脉冲检测准确率（Accuracy）、精确率（Precision）、召回率（Recall）
   - 系统误码率（BER）改善
   - 图像质量指标（PSNR/SSIM）提升
