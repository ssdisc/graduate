# 基于跳频与抗干扰调制的隐蔽图像通信系统设计与实现（论文初稿）

## 中文摘要
随着无线频谱资源日益紧张与电磁对抗环境复杂化，传统图像无线传输方案在可靠性、安全性和低可探测性方面面临多重挑战。针对图像数据在干扰信道中易失真、通信行为易被侦测、以及传统固定阈值抗干扰方法自适应能力不足等问题，本文围绕“跳频抗干扰 + 图像隐蔽传输 + 机器学习辅助接收”开展研究，设计并实现了一套端到端隐蔽图像通信系统。系统发射端将图像经灰度化与变换域压缩后映射为比特载荷，并结合混沌加密与分包传输机制提升保密性和传输鲁棒性；物理层采用跳频与抗干扰调制机制，并结合信道编码、交织与会话/物理头部保护，提高在窄带干扰等场景下的链路可用性；接收端构建同步、干扰抑制、软判决译码与图像重建闭环，并引入基于数据驱动的干扰识别和软擦除策略，增强复杂条件下的恢复能力。  
在 MATLAB 平台上，本文实现了完整仿真链路，并针对不同信噪比点进行方法对比，评估指标包括误码率（BER）、分包错误率（PER）、图像峰值信噪比（PSNR）、结构相似性（SSIM）以及隐蔽相关指标。以窄带干扰场景的代表性实验为例，在 Eb/N0=10 dB 条件下，`fh_erasure` 方法的 BER 由基线 `none` 的 0.1301 降低至 0.0438，且 PER 为 0，对应图像恢复质量保持在较高水平。结果表明，所设计系统能够在干扰环境中实现较稳定的图像隐蔽传输，并为后续工程实现与参数优化提供可复现实验基础。

**关键词** 跳频通信 抗干扰调制 隐蔽图像通信 混沌加密 机器学习

## Abstract
With increasingly congested spectrum resources and more complex electromagnetic confrontation environments, conventional wireless image transmission schemes face significant challenges in reliability, security, and low detectability. To address image distortion in interfered channels, high transmission detectability, and the limited adaptability of fixed-threshold anti-jamming approaches, this thesis proposes and implements an end-to-end covert image communication system by integrating frequency hopping anti-jamming, secure image delivery, and machine-learning-assisted reception.  
At the transmitter side, images are converted into bit payloads through grayscale preprocessing and transform-domain compression, and then protected by chaos-based encryption and packetized transmission. At the physical layer, frequency hopping and anti-jamming modulation are combined with channel coding, interleaving, and protected headers to improve link robustness under narrowband interference. At the receiver side, a closed-loop chain including synchronization, interference mitigation, soft-decision decoding, and image reconstruction is constructed, with data-driven interference identification and soft-erasure strategies introduced to improve robustness under complex conditions.  
An end-to-end simulation platform is implemented in MATLAB. Comparative evaluations are conducted across multiple Eb/N0 points using BER, PER, PSNR, SSIM, and covert-related metrics. In a representative narrowband interference experiment at Eb/N0 = 10 dB, the BER of `fh_erasure` is reduced from 0.1301 (`none` baseline) to 0.0438, while PER remains 0 and image quality stays high. The results indicate that the proposed system can maintain stable covert image transmission in interference-rich environments and provides a reproducible foundation for further engineering refinement.

**Keywords** frequency hopping, anti-jamming modulation, covert image communication, chaos encryption, machine learning

## 第一章 绪论

### 1.1 研究背景
无线图像传输在应急通信、工业监测与特殊场景信息保障中具有重要应用价值，但实际信道普遍存在噪声、窄带干扰与多径效应，导致传统方案在可靠性方面表现受限。另一方面，仅关注“内容保密”已无法满足实际需求，通信行为本身的低可探测性与抗截获能力逐渐成为系统设计重点。因此，面向复杂干扰环境构建兼顾可靠传输、抗干扰能力与隐蔽性的图像通信系统，具有明确的工程意义。

### 1.2 研究意义
本课题将跳频通信、抗干扰调制、图像加密压缩与机器学习方法融合，旨在实现以下目标：
1. 在有限带宽与干扰环境下提升图像传输成功率与重建质量。
2. 通过跳频与加密机制降低通信可探测性与截获风险。
3. 通过数据驱动方法提升干扰检测与接收端自适应能力。

### 1.3 国内外研究现状概述
现有研究主要沿三条路径推进：一是以频率分集与跳频序列优化为核心的抗干扰通信研究；二是面向图像数据的压缩、加密与鲁棒传输研究；三是将深度学习引入物理层检测、均衡与抗干扰决策的研究。已有工作为本文提供了理论基础，但在“端到端闭环系统整合”与“隐蔽通信评估一体化”方面仍有提升空间。

### 1.4 本文主要工作
结合任务书与开题目标，本文完成如下工作：
1. 设计并实现了图像隐蔽传输端到端系统，涵盖图像预处理、混沌加密、分包与物理层传输、接收重建和指标评估。
2. 构建了包含 AWGN、窄带、单音、扫频、脉冲与多径机制的仿真信道，并实现多类抗干扰方法统一比较。
3. 实现机器学习辅助抗干扰模块（含脉冲干扰检测、干扰分类、窄带动作选择、FH 软擦除与多径均衡模型）及离线训练加载流程。
4. 完成了基于 BER/PER/PSNR/SSIM/KL 与 Warden 检测层的综合实验评估。

### 1.5 论文结构
全文共五章：第一章为绪论；第二章介绍系统模型与总体架构；第三章给出关键算法设计与实现；第四章开展仿真实验与结果分析；第五章总结全文并给出后续展望。

## 第二章 系统模型与总体架构

### 2.1 系统设计目标
系统目标包括：  
1. 在窄带等干扰场景下维持较低 BER/PER。  
2. 在保障链路可靠性的同时维持图像恢复质量。  
3. 具备可扩展的隐蔽评估与方法比较能力。  
4. 支持离线训练与在线加载，形成可复现实验流程。

### 2.2 总体流程
系统总体流程为：  
图像输入 -> 预处理与压缩 -> 混沌加密 -> 分包与头部封装 -> 信道编码/交织/调制 -> 跳频发射 -> 干扰信道传输 -> 同步与抗干扰接收 -> 软判决译码与重建 -> 质量与隐蔽性评估。

### 2.3 发射端模型
发射端主要由图像源处理、载荷封装与物理层发送三部分组成：
1. 图像源处理：支持原始与 DCT 载荷编码。
2. 安全处理：支持 Arnold 置乱与混沌扩散，实现图像域或比特域加密。
3. 物理层发送：支持扰码、FEC、交织、调制（默认 QPSK）、跳频与波形成型。

### 2.4 信道与干扰模型
信道侧建立多类可配置干扰：
1. 加性高斯白噪声（AWGN）。
2. 单音干扰与窄带干扰。
3. 扫频干扰与脉冲噪声。
4. 多径与可选瑞利衰落。
通过 Eb/N0 与 JSR 两轴可完成链路性能扫描和方法对比。

### 2.5 接收端模型
接收端包含同步、前端抑制、译码重建三部分：
1. 同步链路：帧同步、分数定时、CFO 补偿、载波 PLL。
2. 抗干扰前端：支持传统方法（陷波、带阻、空白等）与 ML 方法。
3. 重建链路：软判决译码、分包恢复、图像重建与质量评价。

### 2.6 隐蔽评估模型
系统集成 Warden 检测器，支持多层评估：
1. 固定 PFA 目标层（energyNp）。
2. 最优阈值层（energyOpt）。
3. 含不确定性的最优检测层（energyOptUncertain）。
4. 跳频窄带侦测层与循环平稳侦测层。  
该设计用于评估通信行为可探测性与隐蔽性强弱。

## 第三章 关键算法设计与实现

### 3.1 图像处理与混沌加密
为适应带宽约束并增强保密性，系统采用“压缩 + 加密”联合策略：
1. 图像经灰度化及可选 DCT 变换域裁剪形成紧凑载荷。
2. 采用混沌映射生成密钥流，结合置乱与扩散提升抗分析能力。
3. 支持按包独立加密，降低单包错误的扩散影响。

### 3.2 分包传输与头部保护
系统采用分包通信机制，将载荷拆分为多个数据包，并设计会话头/物理头用于接收端恢复配置。结合头部重复与保护编码，提高在干扰条件下的头部可解性，降低“有信号但无法解帧”的风险。

### 3.3 跳频与调制传输机制
物理层采用跳频抗干扰传输。默认慢跳频模式下，每个 hop 承载固定数量符号，实现频域分散。结合 QPSK 调制与波形成型后，信号在带宽利用与抗窄带压制方面取得平衡。

### 3.4 编码与纠错策略
系统在内层采用卷积码/LDPC 可选结构，默认使用 LDPC 短帧与软判决译码。外层支持按包 RS 保护与丢包补偿，面向“整包失败”场景提升图像恢复鲁棒性。

### 3.5 抗干扰方法集成
为进行系统性对比，平台统一集成多类抑制方法：
1. 传统频域或时域抑制：`fft_notch`、`fft_bandstop`、`adaptive_notch`、`blanking`、`clipping` 等。
2. 跳频软擦除策略：`fh_erasure`。
3. 机器学习方法：`ml_cnn`、`ml_gru`、`ml_fh_erasure` 以及自适应前端 `adaptive_ml_frontend`。

### 3.6 机器学习训练与加载流程
系统提供离线训练入口 `run_offline_training.m`，并在 `models/` 下保存模型工件。主仿真入口 `run_demo.m` 根据当前方法配置自动加载所需模型，实现“训练-加载-对比评估”闭环。

## 第四章 仿真平台与实验结果分析

### 4.1 仿真平台与实验设置
本节采用 `results/matlab_20260423_142626` 的完整实验结果进行说明。其关键配置如下：

| 项目 | 配置 |
| --- | --- |
| 调制方式 | QPSK |
| 信道编码 | LDPC（short, rate=1/2） |
| 载荷编码 | DCT |
| Eb/N0 点 | 4, 6, 8, 10 dB |
| JSR | 0 dB |
| 干扰类型 | 窄带干扰（centerFreqPoints=2.5, bandwidthFreqPoints=1） |
| 比较方法 | none, fh_erasure, ml_fh_erasure, ml_cnn, ml_gru, adaptive_ml_frontend |
| 每点帧数 | 5 |
| 接收分集 | 2 路 MRC |
| Warden 主层 | energyOptUncertain |

说明：本组结果主要用于“方法有效性验证与趋势分析”，最终论文结论仍需更大样本量实验补强。

### 4.2 评价指标
本文采用以下指标：
1. BER：比特误码率，衡量译码准确性。
2. PER：分包错误率，衡量业务层可恢复程度。
3. PSNR/SSIM：图像重建质量指标。
4. KL 指标：信号与噪声统计分布差异。
5. Warden 检测指标（Pd/Pfa/Pe）：衡量通信可探测性与隐蔽性。

### 4.3 通信性能结果
在 Eb/N0=10 dB 下，各方法关键结果如下：

| 方法 | BER | PER | payloadSuccessRate | PSNR(comp, dB) |
| --- | ---: | ---: | ---: | ---: |
| none | 0.1301 | 0.0000 | 1.0000 | 30.59 |
| fh_erasure | 0.0438 | 0.0000 | 1.0000 | 30.59 |
| ml_fh_erasure | 0.1290 | 0.0000 | 1.0000 | 30.59 |
| ml_cnn | 0.1585 | 0.0476 | 0.9524 | 29.09 |
| ml_gru | 0.3869 | 0.3810 | 0.6190 | 20.55 |
| adaptive_ml_frontend | 0.1148 | 0.0000 | 1.0000 | 30.59 |

可见在该窄带场景下，`fh_erasure` 的 BER 最低，较 `none` 有明显下降；`adaptive_ml_frontend` 相比基线亦有改进。另一方面，`ml_gru` 在当前参数与样本量条件下表现较弱，说明其训练域与在线场景匹配度仍需进一步优化。

在 Eb/N0 从 4 dB 提升至 10 dB 过程中，KL 对称散度由 7.5086 降至 1.6408，说明高信噪比下信号与噪声统计差异减小，系统进入更稳定工作区间。

### 4.4 隐蔽性评估结果
以主评估层 `energyOptUncertain` 为例，Pe 在 Eb/N0=4/6/8/10 dB 下分别为 0.4853、0.4820、0.4750、0.4673。该结果表明，在当前配置下 Warden 仍具有一定检测能力，但系统具备可量化的隐蔽评估基础，可用于后续参数优化和对抗策略改进。

### 4.5 结果讨论
1. 跳频软擦除在窄带干扰下优势明显，说明“频点级异常识别 + 软可靠度降权”策略有效。
2. 机器学习方法并非必然优于传统方法，其效果高度依赖训练域覆盖、阈值设定和在线场景匹配。
3. 当前结果样本量偏小（每点 5 帧），结论可用于初稿与趋势说明，但不足以支撑终稿中的强统计结论。

## 第五章 总结与展望

### 5.1 全文总结
本文围绕复杂干扰环境下的图像隐蔽通信问题，完成了从系统建模、关键算法设计到仿真验证的完整研究工作。通过将图像预处理与混沌加密、跳频抗干扰传输、编码纠错、接收端抑制以及隐蔽检测评估有机结合，构建了可复现实验平台，并在代表性干扰场景下验证了方案有效性。实验结果表明，所提系统能够在干扰条件下实现较稳定的图像恢复，并具备开展隐蔽性分析与方法比较的能力。

### 5.2 不足与后续工作
后续可从以下方向继续完善：
1. 提升统计置信度：将每点帧数提升至 30-100，并增加随机种子重复实验。
2. 拓展场景覆盖：系统化扫描 JSR、多径与混合干扰组合。
3. 优化学习模型：扩大训练域并改进在线阈值/校准策略，提升模型泛化能力。
4. 完善安全评估：引入 Eve 接收口径对比，形成“可靠通信-抗截获-低探测”三维评估框架。
5. 推进工程实现：结合 SDR 或半实物平台进行实时验证。
